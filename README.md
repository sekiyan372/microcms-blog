## このリポジトリの役割

## 推奨実行環境

- node: v16系

## 実行コマンド

```bash
# 開発環境
$ yarn dev

# ビルド
$ yarn build

# ビルドしたプロジェクトの立ち上げ
$ yarn start

# コードチェック
$ yarn lint

# コードフォーマッタ
$ yarn format
```
